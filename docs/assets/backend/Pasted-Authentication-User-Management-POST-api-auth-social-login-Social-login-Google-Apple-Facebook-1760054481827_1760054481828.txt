Authentication & User Management
POST /api/auth/social-login - Social login (Google, Apple, Facebook)
POST /api/jwt-tokens/logout - Token invalidation on logout
GET /api/auth/profile - Get current user profile
Search & Discovery
GET /api/search?q={query} - Global search endpoint for merchants and commodities
GET /api/merchants/nearby?latitude={lat}&longitude={lng}&radius={radius} - Nearby merchants
GET /api/merchants/nearby/live?lat={lat}&lng={lng}&radius={radius} - Nearby merchants with live locations
Merchants
GET /api/merchants - List all merchants
GET /api/merchants/:id - Get merchant by ID
POST /api/merchants - Create merchant
PUT /api/merchants/:id - Update merchant
DELETE /api/merchants/:id - Delete merchant
GET /api/merchants/:merchantId/commodities - Get merchant's commodities
POST /api/merchants/:merchantId/commodities - Add commodity to merchant
PUT /api/merchants/:merchantId/commodities/:commodityId - Update commodity
DELETE /api/merchants/:merchantId/commodities/:commodityId - Delete commodity
Commodities/Products
GET /api/commodities - List all commodities
GET /api/commodities/:id - Get commodity by ID
Favorites
GET /api/favorites - Get user's favorites
POST /api/favorites - Add favorite (body: {itemId, itemType})
DELETE /api/favorites/:itemId - Remove favorite
Location Services
PUT /api/location/live - Update user's live location (body: {latitude, longitude, timestamp})
GET /api/location/live/:userId - Get user's live location
Transactions & Payments
GET /api/transactions - Get transaction history with filters
GET /api/transactions/:id - Get transaction details
POST /api/transactions/:transactionId/refund - Request refund
GET /api/payment-methods - Get saved payment methods
POST /api/payment-methods - Add payment method
DELETE /api/payment-methods/:id - Remove payment method
POST /api/payment-methods/:id/default - Set default payment method
Toll Payments
GET /api/toll-payments?fromDate={date}&toDate={date}&limit={n}&offset={n} - Get toll payment history
POST /api/toll-payments - Create toll payment
Orders
GET /api/orders - Get user's orders
GET /api/orders/:id - Get order details
POST /api/orders - Create new order
PUT /api/orders/:id - Update order
PUT /api/orders/:id/status - Update order status
DELETE /api/orders/:id - Cancel order
Communication
GET /api/conversations - Get user's conversations
GET /api/conversations/:id - Get conversation details
GET /api/conversations/:id/messages - Get conversation messages
POST /api/conversations/:id/messages - Send message
Admin Endpoints
GET /api/admin/kyc/documents - Get KYC documents for review
POST /api/admin/kyc/approve - Approve KYC document (body: {documentId, userId, notes})
POST /api/admin/kyc/reject - Reject KYC document (body: {documentId, userId, reason, notes})
POST /api/admin/kyc/batch - Batch KYC action (body: {documentIds, action, notes})
Merchant Analytics (if merchantService.getAnalytics exists)
GET /api/merchants/:merchantId/analytics - Get merchant analytics data
Cart Management
GET /api/cart - Get user's cart
POST /api/cart/items - Add item to cart
PUT /api/cart/items/:itemId - Update cart item
DELETE /api/cart/items/:itemId - Remove from cart
DELETE /api/cart - Clear cart
Notifications
GET /api/notifications - Get user notifications
PUT /api/notifications/:id/read - Mark notification as read
POST /api/notifications/register-token - Register push notification token
KYC
POST /api/kyc/documents - Submit KYC document
GET /api/kyc/status - Get KYC verification status
